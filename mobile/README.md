# Aplicacion movil Powerful Control System

Esta carpeta contiene una sola base Flutter para Android e iPhone. La aplicacion
consume exclusivamente la API versionada `/api/v1/` del backend Go; no conoce
PostgreSQL ni replica reglas de negocio empresariales.

El proyecto se encuentra en `powerful_control_system_app/`. Consulte su README
y `documentos/arquitectura_aplicacion_movil.md` antes de agregar un modulo.
