# EmulatorJS

Coloca aqui los archivos oficiales de EmulatorJS, de forma que exista:

```text
emulator/data/loader.js
```

El frontend del emulador esta en `../public/index.html` y carga este `loader.js` cuando el usuario presiona **Jugar**.

En Linux puedes ejecutar desde la carpeta `jeugos`:

```bash
sh deploy/fetch_emulatorjs.sh
```

Tambien puedes descargar manualmente la version oficial y copiar su carpeta `data` dentro de `emulator`.
