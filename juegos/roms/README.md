# ROMs

Copia aqui tus ROMs de SNES en formatos permitidos:

- `.sfc`
- `.smc`
- `.fig`
- `.swc`
- `.zip`

El servidor solo lista y sirve archivos de esta carpeta. No existe subida desde el frontend.

Los juegos que coloques aqui apareceran automaticamente en el emulador web en `../public/index.html` y tambien en el endpoint `/api/roms`.

Usa solo ROMs que tengas derecho legal a utilizar.

Nota local: los archivos de ROM dentro de esta carpeta estan ignorados por git. Para probar el emulador se pueden usar ROMs homebrew/publicas, por ejemplo las de Retrobrews: https://github.com/retrobrews/snes-games.
