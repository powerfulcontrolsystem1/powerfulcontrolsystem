# Documentacion de diagramas

Fecha de actualizacion: 2026-04-07

## Objetivo
Este directorio centraliza los diagramas tecnicos y funcionales del sistema POS multiempresa.

## Archivos incluidos
- `diagrama_arquitectura_sistema.md`: vista de componentes, capas y dependencias externas.
- `diagrama_casos_de_uso.md`: actores y casos de uso principales del sistema.
- `diagrama_entidad_relacion.md`: modelo ER de entidades y relaciones clave.
- `diagrama_flujo_procesos.md`: flujo operativo y rutas criticas por modulo.
- `diagrama_roles_permisos.md`: matriz resumida de permisos por rol.
- `estructura_del_codigo.md`: mapa de estructura tecnica y cambios estructurales por fecha.
- `versionado.md`: historial de versiones de la documentacion de diagramas.
- `regla_agente_go_diagramas.md`: lineamientos de mantenimiento para diagramas.

## Regla de actualizacion
- Actualizar estos diagramas cuando cambien rutas, modelos, tablas, permisos o flujos funcionales.
- Mantener consistencia entre diagramas, `documentos/estructura_bd.md` y `documentos/descripcion_del_proyecto`.
- Registrar cada cambio en `documentos/descripcion_de_archivos`, `documentos/historial_de_cambios` y `versionado.md`.
