Inventario y productos

Tutorial rápido

Resumen práctico para usar productos, bodegas, lotes, precios, recetas y proveedores sin perder el orden operativo.

Cómo organizar el módulo

Productos: crea aquí el catálogo base, servicios y referencias vendibles.

Bodegas: separa inventario por sede, cuarto frío, mostrador o almacén principal.

Lotes: úsalos cuando necesites vencimiento, trazabilidad por compras o retiros selectivos.

Recetas: sirven para descontar ingredientes reales cuando vendes combos, platos o preparados.

Proveedores: administra compras desde la misma estructura del inventario para no duplicar datos.

Flujo recomendado

1. Configura bodegas y categorías antes de cargar productos masivos.

2. Crea productos con impuesto, unidad, costo y precio bien definidos.

3. Si manejas vencimientos, activa lotes desde la compra o la entrada inicial.

4. Si vendes preparados, arma la receta y revisa margen antes de publicarla.

5. Revisa historial de productos para auditar cambios de costo, precio o impuestos.
