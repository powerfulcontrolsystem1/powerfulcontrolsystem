Guía pública rápida

La página pública comparte al cliente el host del servidor, las descargas del cliente y del servidor, además del ID y contraseña del equipo si la sesión fue aprobada.

  1. Configura el host y la clave pública de RustDesk.
  2. Guarda URLs de descarga para cliente y servidor.
  3. Registra el dispositivo con ID RustDesk o con URL embebible.
  4. Crea una sesión y comparte el enlace público generado.
