Soporte remoto por empresa

Este módulo permite visualizar un escritorio remoto dentro del panel usando una URL web embebible (por ejemplo noVNC/Guacamole/RustDesk Web) aislada por empresa.

Recomendación técnica (plugin/agente)

Para operación tipo AnyDesk/TeamViewer simplificada y segura, usa un agente local en cada empresa que publique stream_url embebible (noVNC/Guacamole/RustDesk Web) y envíe heartbeat a /api/public/soporte_remoto?action=heartbeat con empresa_id, codigo_dispositivo y PIN.

El módulo ya soporta aislamiento por empresa_id, token temporal de visualización por sesión y trazabilidad de estado.
