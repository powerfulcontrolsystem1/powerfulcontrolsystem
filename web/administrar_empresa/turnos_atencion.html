Turnos de atención

Cola profesional para bancos, recepción, cajas y servicio al cliente

Emite tickets, organiza servicios, asigna puestos, llama el siguiente turno y alimenta una pantalla pública para TV con el historial de llamados y el estado de la fila.

Operación del módulo

Este flujo está pensado para sucursales, cajas, clínicas, bancos, gimnasios o recepciones donde el cliente toma un ticket y espera a que el sistema lo llame en pantalla.

Salida para pantalla TV

Abre la pantalla de llamados en un televisor, activa el audio y deja visible el turno actual con historial reciente.

1
Abrir pantallaUsa el enlace publico del modulo para llevarlo a un navegador del televisor o un computador conectado por HDMI.
2
Activar sonidoLa TV mostrara un boton para habilitar alarmas. Los navegadores lo piden por seguridad antes de reproducir audio.
3
Llamar turnoCada nuevo llamado resalta la pantalla, reproduce alarma y anuncia turno y puesto cuando el navegador lo permite.
