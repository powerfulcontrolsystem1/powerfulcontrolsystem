Tutorial de funcionamiento

  1. Descarga el cliente RustDesk para el sistema operativo del equipo que recibirá soporte.
  2. Instala el programa y déjalo abierto cuando el operador vaya a conectarse.
  3. Si tu empresa administra un nodo propio, descarga además el servidor para el VPS o el host dedicado.
  4. La activación y prueba del servicio RustDesk del VPS se hace desde configuración avanzada del super administrador.
  5. Conserva estos instaladores como referencia oficial del proyecto para evitar mezclar versiones no validadas.
