Solitario Clásico (Senku)

Instrucciones:

  • Usa las flechas del teclado para mover el cursor rojo.
  • Presiona Espacio para seleccionar una ficha.
  • Mueve el cursor saltando sobre otra ficha hacia un hueco vacío y presiona Espacio.
  • El objetivo es dejar una sola ficha en el centro del tablero.
