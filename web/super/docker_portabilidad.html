Traslado a otro VPS

  1. Instalar Docker Engine y Docker Compose v2 en la nueva VPS.
  2. Descomprimir este paquete en /root/powerfulcontrolsystem.
  3. Crear deploy/.env.platform desde la plantilla y cargar secretos reales fuera del paquete.
  4. Restaurar snapshots de PostgreSQL y volumenes persistentes si se migra produccion.
  5. Ejecutar bash deploy/scripts/vps-docker-preflight.sh.
  6. Levantar con docker compose --env-file deploy/.env.platform -f deploy/docker-compose.platform.yml up -d --build.
  7. Para publicar TLS desde Docker, ejecutar CONFIRM_DOCKER_EDGE=YES bash deploy/scripts/vps-docker-edge-up.sh despues de apuntar DNS.

Politica de seguridad

La descarga esta pensada para soporte y migracion tecnica. No reemplaza el backup de datos.

Snapshot completo VPS
