Base de datos superadministrador (chat global)
El chat global siempre incorpora, en cada consulta, el inventario de todas las tablas visibles en el esquema: número de filas, columnas (nombre y tipo) y reparto de administradores por rol. No se envían valores de fila, contraseñas, tokens ni contenidos de configuración sensibles.
El servidor solo adjunta SELECT y agregados aprobados (listado de empresas, muestras de productos, finanzas, clientes, ventas). Prohíbe a la IA sugerir ediciones o PCS_ACTION de negocio; el prompt refuerza solo análisis y consulta informativa.
