Editor de tarjetas

La cantidad se aplica al cambiar este campo y se guarda junto con toda la configuración.

Tipos de tarjeta: por defecto se usa "tarjeta información más foto". Si eliges "tarjeta banner", el index mostrará solamente la imagen ocupando toda la tarjeta.

Medidas recomendadas para banner: relacion 4:3. Pequena: 640 x 480 px, mediana: 800 x 600 px, grande: 1000 x 750 px. Tambien puedes usar una imagen un poco mas pequena con la misma proporcion para evitar recortes.

Cada tarjeta define imagen, título, descripción breve, destino de Probar Gratis y el contenido ampliado que se mostrará en la landing descriptiva. Los ajustes visuales aplican de forma global al index y a la pagina descriptiva.
