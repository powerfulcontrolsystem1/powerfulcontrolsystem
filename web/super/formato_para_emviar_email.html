Formatos para enviar email

Administra desde un solo lugar los correos que salen del sistema. Incluye confirmación de correo, pago de licencias y plantillas recomendadas para recuperación y alertas operativas.

Cargando plantillas...

Submenú

Cada botón abre un grupo de formatos.

Variables disponibles: escríbelas tal como aparecen entre llaves dobles, por ejemplo {{name}} o {{confirm_url}}.

Correo cuando se confirma cuenta administrativa

Se usa en el registro del super o administradores del panel.

{{name}} {{confirm_url}} {{login_url}}
--

Correo cuando se confirma cuenta de usuario de empresa

Este es el formato principal para los usuarios creados por cada empresa. Aquí puedes editar el mensaje que recibe el usuario antes de confirmar su correo.

{{name}} {{company_name}} {{confirm_url}} {{login_url}} {{admin_message}} {{admin_message_block_text}} {{admin_message_block_html}}
--

Correo de invitación para compartir empresa entre administradores

Se envía cuando un administrador propietario comparte su empresa con otro administrador ya registrado. El invitado acepta desde el enlace y la empresa aparece en su selector.

{{name}} {{company_name}} {{invited_by_name}} {{accept_url}} {{login_url}} {{admin_message}} {{admin_message_block_text}} {{admin_message_block_html}}
--

Correo de bienvenida cuando un cliente paga una licencia

Se envia cuando el pago queda aprobado y la licencia se activa. Funciona como bienvenida a PCS y puede adjuntar la factura electronica en PDF segun la configuracion de Licencias; la licencia del software se descarga desde Administrar empresa.

{{company_name}} {{license_name}} {{provider}} {{reference}} {{license_name_line}} {{start_date_line}} {{end_date_line}} {{reference_line}}
--

PDF de licencia del software

Este texto se convierte en el PDF que el cliente descarga desde Administrar empresa > Licencia del sistema. Ya no se adjunta al correo de bienvenida.

{{company_name}} {{company_nit}} {{license_name}} {{license_code}} {{provider}} {{reference}} {{license_download_url}} {{issue_date}} {{start_date}} {{end_date}} {{amount_paid}} {{system_name}}
