Ayuda - Chat con IA

Guia para usar el asistente IA con contexto de negocio, auditoria, ayuda del sistema y datos seguros del backend.

Como usarlo bien

  1. Escribe la tarea con contexto: empresa, modulo, fecha, cliente, estacion, producto o documento esperado.
  2. Si quieres datos reales, pide una consulta: "revisa los ultimos carritos pagados" o "busca productos sin bodega".
  3. Si quieres crear o cambiar datos, la IA debe proponer una accion y pedir confirmacion antes de ejecutar.
  4. Para eliminar, el sistema debe pedir confirmacion adicional. No aceptes eliminaciones sin revisar respaldo o auditoria.

Contexto que usa la IA

Auditoria en tiempo real

La IA recibe eventos recientes, busquedas profundas y resultados seguros preparados por el backend.

Logica del negocio

Super Administrador puede editar el contexto canonico en /super/contexto_ia_logica_negocio.html.

Ayuda del sistema

La IA tambien recibe un resumen de estas paginas de ayuda para responder alineada con la documentacion visible.

Datos por empresa

En chat empresarial solo debe usar informacion de la empresa activa y respetar empresa_id.

Acciones y confirmaciones

  • GET/OPEN: consultar datos o abrir una vista.
  • POST/PUT: crear o actualizar datos, siempre con confirmacion.
  • DELETE: eliminar datos, con confirmacion adicional y advertencia del impacto.

Recomendacion: pide primero una consulta para revisar resultados y luego confirma cambios.

Chat IA empresarial

  • Opera solo sobre la empresa seleccionada.
  • Puede ayudar con carrito, productos, inventario, estaciones, caja, reportes, facturacion, venta publica, tareas y agenda.
  • Puede proponer acciones CRUD por empresa si el usuario tiene permisos.
  • Todas las acciones confirmadas deben dejar trazabilidad.

Chat IA global

  • Sirve para soporte de plataforma, licencias, tipos de empresa, configuraciones globales, pagos e integraciones.
  • Puede analizar datos agregados entregados por el backend, sin mezclar empresas fuera de permisos.
  • Cuando proponga acciones, siempre requiere confirmacion.

Voz, robot y secretaria

Chat cuadrado

Ventana compacta tradicional con mensajes, adjuntos, microfono, modo y herramientas de documentos.

Robot o secretaria 3D

La conversacion fluye en globos junto al avatar. El boton stop detiene la voz solo por el momento.

Voz rapida

Usa la voz del navegador/computador. Puede ser femenina o masculina y responde mas rapido.

Voz natural VPS

Usa Piper/FastAPI. Si esta desactivada o falla, el chat cae a voz del navegador o texto.

Documentos desde el chat

  1. Pide el documento con datos suficientes: tipo, cliente, fecha, productos, valores y formato esperado.
  2. La IA debe preguntar solo datos indispensables que falten.
  3. Cuando el documento esta listo, el sistema muestra exportacion a PDF, Word/DOCX, Excel/XLSX, TXT o JSON cuando aplique.
  4. Si falla un conversor, el chat no debe romperse: debe mostrar mensaje claro y permitir otro formato.

Ejemplos utiles

  • "Genera un Excel con productos, solo nombre y precio".
  • "Revisa por que no puedo agregar este producto al carrito".
  • "Activa el robot y activa tu voz".
  • "Muestrame los movimientos de caja de hoy y prepara un reporte".
  • "Explica los pasos para configurar DIAN en esta empresa".
