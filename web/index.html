Módulo 1 — Autenticación (Login con Google)

Descripción: Inicio de sesión mediante cuenta de Google (OAuth2). La implementación backend en Go maneja el flujo y crea el usuario en la base de datos con rol por defecto administrador.

Acción:

Iniciar sesión con Google

Notas de implementación:

  • El backend intercambia el código por token y obtiene userinfo.
  • Usuarios nuevos obtienen el rol administrador automáticamente.
  • Al completar, el usuario es redirigido a seleccionar_empresa.html.
